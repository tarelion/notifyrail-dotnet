# NotifyRail README Hero Source

NotifyRail is a learning-focused C#/.NET backend that simulates reliable
notification delivery.

Core proof points:

- Message intake
- PostgreSQL-backed delivery queue
- Hosted background worker
- Retry/backoff
- Provider callbacks
- OTP verification
- Delivery reports
- Tests: 70 passed, 0 failed

Hero intent:

- README opening image
- Fewer modules than the system overview infographic
- True wide hero composition
- Large project name
- Pipeline visual
- Test badge
- Minimal text
